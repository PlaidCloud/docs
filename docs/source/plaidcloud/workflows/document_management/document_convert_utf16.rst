Description
-----------

Updates file encoding and converts all characters to UTF-16. This is
particularly useful if the information source has mixed encodings or
other tools don't support certain encodings.

Examples
--------

Select the input file and browse for the file within that location. Select the desired output
location, and browse to selected the desired location for the file. Save and run. |Document Convert File to UTF-16
Encoding|
