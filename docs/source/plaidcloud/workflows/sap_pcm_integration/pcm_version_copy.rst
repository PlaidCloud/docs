Description
-----------

Copies an SAP Profitability and Cost Management (PCM) model version
within the same model.

Our Credentials
---------------

Tartan Solutions is an official SAP Partner and a preferred
vendor of services related to SAP PCM model design and implementation.

|SAP Partner|


Workflow Configuration Forms
----------------------------

Examples
--------

Select Agent to Use from the dropdown, enter "Model Name", "Origin Period Name", and "Destination Period Name" in the "Model Information" field. Click the "Wait for Copy to Complete" check box, then click "Save and Run Step".
